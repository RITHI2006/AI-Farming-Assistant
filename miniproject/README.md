# Farmer Assistant - Smart AI Farmer Assistant

A comprehensive React.js web application built with Vite and Tailwind CSS for modern farming assistance. This application provides AI-powered features including voice assistance, disease detection, weather monitoring, market prices, government schemes, and community forums.

## Features

- 🎤 **Voice Assistant** - Ask questions in Tamil/Regional languages (Bhashini API integration pending)
- 🔍 **Disease Detector** - Upload crop leaf photos for instant disease detection (TensorFlow Lite integration pending)
- ☁️ **Smart Weather** - Real-time temperature, humidity, and actionable alerts
- 📊 **Mandi Live** - Real-time market prices for crops (Tomato, Onion, Rice, etc.)
- 📋 **Sarkaar Schemes** - Searchable list of government agriculture subsidies
- 💬 **Community Forum** - Connect with fellow farmers and share experiences
- 💚 **WhatsApp Integration** - Quick access to WhatsApp Business API
- 📱 **Offline Indicator** - Status for local AI model loading

## Tech Stack

- **React.js** - UI Framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Installation

After cloning the repository, install the dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```
